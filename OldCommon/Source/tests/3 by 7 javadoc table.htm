Attribute Type Description
============================= ================== =================================
target ArrayList List of {@link XactXlatBean} objects as targets. Only one one element should exist
offsets ArrayList List of {@link XactXlatBean} objects as offsets. At least one element is required.
targetTypes ArrayList List of applicable target transaction objects of either type {@link GlAccountTypeCatg} or {@link GlAccounts}
offsetTypes ArrayList List of applicable offset transaction objects of either type {@link GlAccountTypeCatg} or {@link GlAccounts}
message String Message denoting the result of the operation.
