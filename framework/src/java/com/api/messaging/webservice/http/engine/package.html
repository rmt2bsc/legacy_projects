API providing classes and interfaces to operate the HTTP URL Query-Based Web Service engine.

How to use
Add the servlet mapping to the web.xml configuration file of the Service Dispatch application as such: 
	      <servlet>
          <servlet-name>remoteservices</servlet-name>
          <servlet-class>com.api.messaging.webservice.http.engine.HttpServiceController</servlet-class>
        </servlet>
  
			  <servlet-mapping>
			    <servlet-name>remoteservices</servlet-name>
			    <url-pattern>/remoteservices</url-pattern>
			  </servlet-mapping>
