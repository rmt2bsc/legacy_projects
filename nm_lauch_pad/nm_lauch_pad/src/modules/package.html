This package contains all of the Launch Pad application modules which is technologically powered by Swing and Core Java. Provides classes and interfaces that divides the Launch Pad application in to separate working modules. Launch Pad is comprised of the the following modules:
  • User Administration
  • Price Change Administration
  • Price Audit Administration
  • SI Transfers Administration
  • BI Tranfers Administration
  • SI RTV Administration
  • BI RTV Administration
  • Print Reprint Reports
  • Counts Administration
  • IDT Administration
  • Load Telnet Hex File
